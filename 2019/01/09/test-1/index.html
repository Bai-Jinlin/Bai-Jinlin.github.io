分区教程

使用软件分区助手

首先点击这里下载软件分区助手，安装完成之后在开始菜单里找到并且启动


然后右键Windows的分区->调整大小

点击确定，之后点击左上角的提交

安装步骤重启之后就成功了，但是在Windows10 1804版本可能会提示无法调整Windows系统盘的分区

使用Linux live系统调整分区

点击这里下载镜像文件，再点击这里下载U盘制作工具之后按照以下步骤制作启动U盘

  1. 插入U盘到电脑的USB接口中。
  2. 打开深度启动盘制作工具。
  3. 选择下载的镜像文件以及分区。
  4. 点击开始即可制作。

随后需要从U盘启动，您需要先进入电脑的BIOS界面将U盘设置为第一启动项。台式机一般为 Delete 键、笔记本一般为 F2 或 F10 或 F12 键，即可进入 BIOS 设置界面，具体可以百度自己机器的型号。
进入会看到以下界面

点击最下面的磁盘图标

进入后右键Windows所在的分区，点击卸载

卸载完成后点击右键调整分区

之后就可以调整大小
